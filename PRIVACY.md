# Privacy Policy

Last updated: July 18, 2026

Simple Breath does not collect, transmit, sell, or share personal data with the
app developer or any third party.

When a breathing session starts, the app can enable the watch's heart-rate
sensor and create a standard Garmin FIT activity containing session duration
and available heart-rate data. This data remains under the user's Garmin device
and account controls and can be synchronized by Garmin software in the same way
as other recorded activities. Simple Breath does not use network access and
does not operate an external server.

The app stores its configuration locally through the standard Garmin Connect IQ
settings system. These settings include breathing durations, session duration,
circle color, vibration mode, and sound mode.

Users can remove saved activities through Garmin Connect or their Garmin device
and can uninstall the app at any time. Questions about this policy can be
submitted through the project's public GitHub issue tracker.
